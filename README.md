# automated-reporting-system
Автоматизированная система генерации PDF отчетов из SQL данных системы видеонаблюдения
- Ежедневные отчеты по состоянию камер видеонаблюдения.
- Автоматическая отправка сгенерированных отчетов по email.
- Отправка сообщений в Telegram о запуске выполнения скрипта, о результатах генерации отчетов, и результатов отправки их по адресам email.
- Генерация отчетов для нескольких групп пользователей(у разных групп пользователей доступ к различным группам камер).
