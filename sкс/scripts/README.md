# Скрипты генерации CSV

## Назначение
SH скрипты в этой папке генерируют CSV файлы из данных SQL баз.

## Структура скриптов
Каждый скрипт должен:
1. Подключаться к соответствующей БД
2. Выполнять SQL запрос для получения данных
3. Сохранять результат в CSV с разделителем `;`

## Пример скрипта
```bash
#!/bin/bash
# generate_example_csv.sh
mysql -u user -p database -e "SELECT * FROM cameras" | tr '\t' ';' > cameras_oks.csv
